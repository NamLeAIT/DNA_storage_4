Main changes:
1) More compression tools and broader data-type coverage

2) Compression flow simplified to auto-only
   - Compression tab now effectively uses only Auto Fixed and Auto Benchmark
   - Auto routing is Mode-3 only for compression workloads

3) Benchmark metadata added to reports and Analysis tab
   - report["analysis"] with selected_method, score_metric, candidates_tested
   - report["benchmark"] with full candidate ranking and Top 3
   - Analysis tab shows Benchmark Results block and Top 3 table
